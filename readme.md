## 项目技术栈

## 启动方式
项目是根据视频进行分阶段实现，最终版本为`react-enjoy-master`

开启客户端：
`cd react-enjoy-master`
`npm install`
`npm start`

开启服务器端：
`cd react-enjoy-master/mock`
`node index.js`
